# Readme-generator

## Description

A professional README file generated by using inquirer.

## Installation

```
git clone https://github.com/keyurmist/Readme-generator.git
cd Readme-generator
npm install
```

# Getting Started

In order to run the application do the following:

```
npm run start
```
